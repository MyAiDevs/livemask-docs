# LiveMask 客户端开发文档 v3.6（最终闭环版）

## 1. 安全相关新增功能

### 1.1 本地黑白名单 + 威胁情报匹配
- `BlacklistManager` 类负责本地缓存 + 快速匹配（支持 IP、Domain、CIDR）
- 连接前必须调用 `isBlocked()` 和 `isThreat()`
- 命中黑名单或威胁情报 → 立即阻断 + 弹出高危安全公告

### 1.2 威胁狩猎提示（用户可控）
- 当后台 `threat_hunting.enabled = true` 且用户未手动关闭时，首页显示橙色叹号警告
- 点击叹号可查看威胁情报最后更新时间
- 用户可手动关闭（本地优先 + 同步后端记录）

### 1.3 配置热更新 + 自动回滚
- 所有配置拉取时校验 `config_hash`
- 不一致时自动回滚到上一个可用版本并提示用户

## 2. 留存与忠诚度展示
- 首页显示当前忠诚度等级徽章 + Streak 进度
- 个人中心展示预计下月奖励和续费折扣

## 3. sing-box 集成
- 支持动态注入黑白名单和威胁情报 rule_set
- 支持配置热更新后重新加载部分配置
