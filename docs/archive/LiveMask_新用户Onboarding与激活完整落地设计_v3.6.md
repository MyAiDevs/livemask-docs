# LiveMask 新用户 Onboarding 与激活完整落地设计 v3.6

**创建日期**：2026-05-10  
**优先级**：P0（留存关键）

## 1. 完整 Onboarding 流程

### 阶段划分
1. **注册后引导**（0-5 分钟）
2. **客户端下载与安装**（5-15 分钟）
3. **首次连接成功**（激活核心）
4. **深度使用引导**（首周激活任务）

### 关键激活指标（Activation Metrics）
- D1 激活率：注册后 24 小时内成功连接节点
- D7 激活率：首周内完成至少 3 次连接 + 查看流量统计
- 首周节点切换成功率

## 2. 智能节点推荐策略

```go
// 伪代码

    // 3. 结合实时质量分 + 负载筛选 Top N

```

### 前端实现建议
- 注册成功后自动弹出「推荐节点」弹窗
- 一键连接最优节点按钮
- 支持手动切换 + 智能推荐切换

## 3. 首周激活任务体系（Gamification）

推荐设计 5 个轻量任务：
1. 成功连接一个节点
2. 切换到另一个节点
3. 查看个人流量统计
4. 开启「威胁狩猎」功能（如果有）
5. 完成一次速度测试

完成任务给予积分或小额流量奖励，提升激活趣味性。

## 4. 技术实现要点
- 埋点事件：`user_registered`, `client_downloaded`, `first_connection_success`, `task_completed`
- 与 Feature Flag 联动：新用户默认开启简化版 Onboarding
- 与客户支持系统联动：激活失败自动创建低优先级工单
