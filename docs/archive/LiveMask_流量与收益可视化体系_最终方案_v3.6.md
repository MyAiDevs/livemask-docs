# LiveMask 流量与收益可视化体系 最终方案 v3.6

**版本**：v3.6（最终整合版）  
**日期**：2026-05-10  
**状态**：生产级可落地

---

## 一、整体目标

构建一套**从历史流量分析 → 实时节点监控 → 国家维度洞察 → 收益与质量联动**的完整可视化与决策支持体系，支撑 Admin 运营、商业分析与 Sponsor 公平性管理。

---

## 二、核心数据底座（已完成）

| 数据表                    | 作用                           | 更新频率     | 与可视化关系 |
|---------------------------|--------------------------------|--------------|--------------|
| `node_daily_traffic`      | 节点每日流量聚合               | 每日凌晨     | 收益计算 + Sponsor 分析 |
| `daily_country_traffic`   | 国家每日流量（GeoIP 标记）     | 每日凌晨     | 全球地图 + 国家趋势 |
| `node_quality_scores`     | 节点质量分（多维度）           | 每分钟/小时  | 质量热力 + 收益加权 |
| `sponsor_revenues`        | 赞助商每日收益                 | 每日         | 收益趋势 + 大使贡献 |
| `affiliate_commissions`   | 推广大使佣金                   | 每月         | 大使看板 |

## 三、核心可视化页面（已设计完成）

### 1. 全球流量总览页（`/admin/dashboard/traffic-global`）
- KPI 卡片（总流量、活跃国家、在线节点、峰值带宽）
- 全球流量趋势 Area 图（双轴）
- 世界 Choropleth 热力图
- 国家流量排行表格 + Drill-down

### 2. 国家流量详情页（`/admin/traffic/country/{code}`）
- 该国家详细趋势 + 与全球对比
- Top 节点列表
- 质量分分布

### 3. 赞助商流量与收益分析页
- 双轴图（流量 vs 收益）
- 散点图（质量分 vs 收益）
- 转化漏斗（流量 → 收益 → 实际到账）

### 4. 实时流量监控页（WebSocket）
- 活跃节点世界地图（颜色编码在线/降级）
- 实时节点状态面板
- 异常节点告警列表

---

## 四、实时 WebSocket Hub 架构（已实现）

- **技术栈**：Go + Gorilla WebSocket + Redis Pub/Sub + JWT
- **频道**：
  - `realtime:node_status`（节点在线/降级/流量）
  - `realtime:country_traffic`（国家实时聚合）
  - `realtime:alerts`（P0 异常告警）
- **权限**：Admin 全权限，Sponsor 仅看自己节点
- **完整代码**：已写入 `LiveMask_实时WebSocketHub_Go实现_v3.6.md`

---

## 五、每日聚合任务体系（已实现）

- `node_daily_traffic` 每日刷新任务
- `daily_country_traffic` 每日聚合任务（含 MaxMind GeoLite2）
- 收益计算定时任务（Sponsor + 推广大使）
- 质量评分定时任务

所有任务均支持**幂等重跑任意历史日期**，并与申诉追溯调整联动。
