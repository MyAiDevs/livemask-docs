# LiveMask Icons

Recommended icon metaphors:

| Function | Icon direction |
| --- | --- |
| Connect | power |
| Secure | shield |
| Region | globe |
| Node | server |
| Speed | gauge |
| Subscription | badge / credit card |
| Settings | gear |
| Feedback | message circle |
| Warning | triangle alert |
| Audit | history / file check |

Avoid:

- skulls
- hacker masks
- aggressive threat imagery
- meaningless network lines
