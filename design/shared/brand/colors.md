# LiveMask Colors

Initial color direction:

| Token | Purpose | Direction |
| --- | --- | --- |
| `primary` | Main action and selected state | deep teal / cyan-green |
| `success` | Connected / healthy | green |
| `warning` | Degraded / pending | amber |
| `danger` | Failed / blocked | red |
| `surface` | App background | off-white / near-black |
| `ink` | Primary text | near-black / near-white |
| `muted` | Secondary text | neutral gray |

Final color values should be added after the first Atoms design review.
