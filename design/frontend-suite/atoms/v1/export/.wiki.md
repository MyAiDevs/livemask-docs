# Project Summary
The LiveMask project is a comprehensive web-based backend admin dashboard and public website designed for managing and monitoring secure network access and privacy-focused services. It enables administrators to efficiently oversee user accounts, node health, payment processing, and configuration management, while also providing a seamless experience for potential users through the public website that highlights the product's benefits, pricing, and support.

# Project Module Description
- **Dashboard Overview**: Displays real-time metrics such as active users, node health, payment success rates, and user feedback.
- **Users Management**: Allows administrators to manage user accounts, including searching and filtering based on status and subscription plans.
- **Nodes Management**: Monitors the health status and performance of system nodes, providing insights into load and connectivity.
- **Payments Management**: Tracks payment orders through their lifecycle stages and provides financial analytics.
- **Config Management**: Manages system configurations with version tracking and approval workflows.
- **Audit Logs**: Maintains a complete audit trail of actions performed by users and the system.
- **Feedback Queue**: Collects and manages user feedback, categorizing it based on priority and status.
- **Public Website**: Provides information on product features, pricing, download options, and support resources.

- **src/pages/UsersPage.tsx**: User management interface with filtering and actions.
- **src/pages/NodesPage.tsx**: Node management interface with operational actions.
- **src/pages/PaymentsPage.tsx**: Payment management interface with order tracking.
- **src/pages/ConfigPage.tsx**: Configuration management interface with version control.
- **src/pages/AuditLogsPage.tsx**: Audit log viewer displaying system actions.

- **src/pages/WebsitePage.tsx**: Public website showcasing product features and pricing.

- **src/lib/mock-data.ts**: Mock data for testing and development.

# Technology Stack
- **React**: Frontend framework for building user interfaces.
- **TypeScript**: For type safety and better development experience.
- **Vite**: Build tool for fast development and bundling.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Recharts**: Charting library for data visualization.
- **TanStack Query**: Data fetching library for managing server state.

# Usage
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Run the application:
   ```bash
   pnpm run start
   ```
