# Project Context

## Project Overview
LiveMask Backend Admin Dashboard - A comprehensive web-based admin panel for managing the LiveMask backend system. Features include real-time metrics, user management, node health monitoring, payment tracking, config management, audit logs, and feedback queue.

## Constraints
- Dark theme by default using shadcn dark mode CSS variables
- Color Palette: Dark navy background (#0a0e1a), card backgrounds (#111827), accent blue (#3b82f6), success green (#10b981), warning amber (#f59e0b), destructive red (#ef4444)
- Typography: Inter font family, clean and modern
- Layout: Fixed sidebar navigation + main content area with responsive design
- Charts: Recharts library for data visualization
- All data is mock/demo data for the dashboard MVP
