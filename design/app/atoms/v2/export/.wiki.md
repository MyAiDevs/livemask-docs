# Project Summary
LiveMask is a premium privacy VPN and secure network access web application that provides users with stable and low-friction VPN connectivity across various regions. It features connection management, node selection, subscription plans, diagnostics, and user settings for a seamless experience. The application is designed to feel secure and trustworthy, with a modern interface that includes a dark mode toggle for enhanced usability. Recently, a real-time speed test feature was added to the Home screen, allowing users to measure download/upload speeds through the selected VPN node.

# Project Module Description
- **Auth**: Manages user login and signup via Atoms Cloud.
- **Home**: Displays connection status, allows users to connect/disconnect, and includes a real-time speed test feature.
- **Nodes**: Lists available VPN nodes with search and filter options.
- **Plan**: Displays subscription plans and allows upgrades.
- **Profile**: Manages user settings, including a dark mode toggle and diagnostics.
- **Onboarding**: Guides new users through the app features.
- **Splash**: Initial loading screen with session checks.
- **Diagnostics**: Allows users to report issues with their connection.

# File Description Inventory
- **app/backend/main.py**: Entry point for the backend application.
- **app/backend/core/**: Contains core functionalities such as authentication and database configurations.
- **app/backend/models/**: Database models for various entities like users and subscriptions.
- **app/frontend/src/**: Contains the React source code for the frontend, including pages and components.
- **app/frontend/src/lib/api.ts**: API client to interact with the backend services.
- **app/frontend/src/lib/theme-store.ts**: Manages theme preferences and persistence.
- **app/frontend/src/components/livemask/DesktopSidebar.tsx**: Sidebar component for desktop navigation.
- **app/frontend/src/components/livemask/AppLayout.tsx**: Layout component that conditionally renders the sidebar or bottom navigation based on screen size.
- **app/frontend/src/components/livemask/SpeedGauge.tsx**: Canvas-based animated gauge for displaying download/upload speeds.
- **app/frontend/src/components/livemask/SpeedTestPanel.tsx**: UI component for conducting and displaying results of the speed test.

# Technology Stack
- **Frontend**: React 18, TypeScript, Vite, shadcn/ui, Tailwind CSS
- **Backend**: Atoms Cloud (Auth, Database, Edge Functions)
- **State Management**: Zustand
- **Routing**: React Router

# Usage
1. Install dependencies:
   - For the backend: Use `pip install -r requirements.txt`
   - For the frontend: Run `pnpm install`
   
2. Build the frontend:
   - Navigate to the frontend directory and run `pnpm run build`

3. Start the backend:
   - Run the backend application using `python main.py`

4. Access the application via your web browser.
