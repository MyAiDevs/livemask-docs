# LiveMask App Design

This directory stores App client design artifacts for mobile and desktop
clients.

Primary design brief:

```text
docs/app/LIVEMASK_APP_DESIGN_BRIEF_FOR_ATOMS.md
```

Current design version:

```text
design/app/atoms/v2
```

Current Atoms export:

```text
design/app/atoms/v2/export/
```

This export came from the local `LiveMask App Design Brief` folder and contains
cross-platform App UI code and files for mobile and desktop clients.

When Atoms generates a new App UI version, create the next version folder:

```text
design/app/atoms/v3/export/
design/app/atoms/v3/screenshots/
```

Required client design targets:

```text
iOS
Android
macOS
Windows
Linux
```

Desktop designs must include:

- main window
- compact quick-connect window
- macOS menu bar state
- Windows tray state
- Linux tray/status fallback state
- permission/helper recovery states
